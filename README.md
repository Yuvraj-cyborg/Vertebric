# Vertebric

Vertebric is an AI orchestrator and a toy implementation of an autonomous coding agent. 

The name **Vertebric** comes from "vertebrate". Since Anthropic sounds like "Arthropoda" (which are invertebrates), this project acts as the backbone—the vertebrate—for AI coding tasks.

Designed to be lightweight and fast, Vertebric reads code, writes files, runs commands, and acts as your terminal-based AI assistant.

For usage instructions, see [USE.md](USE.md).
